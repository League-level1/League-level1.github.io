Introduction to Constructors

Objects have a name, member variables, and methods. Before you can use an object in your program, you usually have to create an instance of it. This is done using a constructor. Constructors are explained in the following video: Java Constructors Tutorial

       void sayHi(){
             System.out.println("The platypus " + name + " is smarter than your average platypus.");
       }
}
